# Tech Stack

- Frontend: Next.js 16, React 19, TypeScript 5, Tailwind CSS 4
- UI libraries: Radix UI, Headless UI, Lucide Icons
- State: discriminated unions with pure reducers
- Data: Dexie (IndexedDB), PocketBase (backend)
- Build: Turborepo, pnpm workspaces, Biome (linter/formatter)
- Testing: Vitest (not currently configured with tests)
